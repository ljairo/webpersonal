Aplicativo Web Gestión de Ambulancias - EMED

Este proyecto se centra en un aplicativo web diseñado para gestionar y reportar el uso de ambulancias y el personal asignado a ellas. Diariamente, las 59 microredes de la región Huaura deben registrar sus salidas en este sistema.

Interfaz del usuario

El interfaz de usuario del aplicativo web incluye:

  • Formulario de Registro: Permite a las microredes ingresar los detalles de cada salida de ambulancia, incluyendo la fecha, hora, personal involucrado, y motivo de la salida, entre otros.
  • Informe de Reportes: Presenta un resumen detallado de todos sus reportes registrados, ofreciendo una visión general del uso de las ambulancias y el desempeño del personal a lo largo del tiempo.

Interfaz del Administrador

Interfaz de Reportes

En esta sección, se presentan los 59 reportes correspondientes a cada una de las microredes. Cada reporte muestra la información detallada de las salidas de ambulancia registradas. Además, se indica si el reporte fue completado a la hora establecida o si fue llenado posteriormente.

Interfaz de Gráficos Estadísticos

  • Barra de Porcentaje: Muestra el porcentaje de microredes que han completado su registro diario, ofreciendo una rápida visualización del progreso general.
  • Gráfico de Barras: Detalla la cantidad de personal de salud requerido en las salidas de ambulancia de ese día, proporcionando una visión de las necesidades de recursos humanos.
  • Línea de Tiempo: Presenta un registro temporal de las salidas de ambulancia, indicando cuántos registros se han realizado a lo largo del día.

Interfaz de Administración de Usuarios

Este interfaz permite a los usuarios con permisos adecuados crear nuevas cuentas de usuario para las microredes, facilitando la gestión y asignación de accesos a cada entidad correspondiente.
